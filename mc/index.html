README caffeine-Ink / Salmonized Workspace

Salmonized Workspace 是一个 Minecraft Java Edition 服务器，目前使用的服务端版本是 1.21.8。

"非常朴素的生存服务器，目前还没有什么东西，总之欢迎来玩"

服务器地址：

mc.caffeine.ink

注意：目前本 Minecraft 服务器运行在一台松下笔电上，使用 Cloudflare Tunnel 内网穿透服务。目前已无法直接通过在多人游戏中添加 mc.caffeine.ink 的方式进入服务器。请参阅下文的 "向导：如何连接服务器"。

"一方面是出于安全考虑，因为之前被 Mountains of Lava 炸过服，以及，主播不想给阿里云续费了"

本服务器目前加载的模块有:

AuthMe              #用于验证玩家身份，用法参见向导
SkinsRestorer       #用于自定义皮肤，用法参见向导
SinglePlayerSleep   #单个玩家睡眠即可跳过夜晚或雷暴
Skript              #运行自定义脚本，目前用于 Daily Rewards 机制

向导：安装游戏

你可以使用 Microsoft Windows / macOS / GNU Linux / Android 游玩。

对于 Windows / macOS / GNU Linux 用户，可以使用 HMCL 启动器或其他启动器安装 Java 版 1.21.8 进行游玩。HMCL 是一个开源的 Minecraft Java Edition 启动器，你可以通过下面的链接访问 HMCL 官网，下载稳定版 HMCL，或是查看 HMCL 的官方帮助文档。

HMCL GitHub 项目地址

对于 Android 用户，可以使用 FCL 启动器安装 Java 版 1.21.8 进行游玩。

这些启动器都会自动安装合适的 Java 环境。

本服务器没有开启正版验证（online mode），因此你可以使用离线账号加入服务器游玩。

向导：如何连接服务器

由于本服务器运行在内网穿透环境下，目前无法通过直接在多人游戏中添加 mc.caffeine.ink 的方式进入服务器。你需要先在本地运行 Cloudflare Tunnel 客户端 cloudflared，来建立到服务器的隧道连接。

首先，前往 cloudflared 下载页面，向下滚动到最新一个版本的 Assets，根据你的操作系统下载并安装 cloudflared。

比如，对于 Windows x64 用户，你需要下载：

cloudflared-windows-amd64.msi

对于 macOS 用户，你需要下载：

cloudflared-amd64.pkg      #Intel Mac
cloudflared-arm64.pkg      #Apple Silicon

双击安装，安装完成后，打开 Windows 终端（PowerShell 或 cmd 都可以）或是 macOS 终端，运行：

cloudflared --version

如果返回了类似:

cloudflared version 2025.10.0 (built 2025-10-14T16:16 UTC)

则说明 cloudflared 已经安装成功了。

然后运行如下命令来启动隧道连接：

cloudflared access tcp --hostname mc.caffeine.ink --listener 127.0.0.1:25565

如果一切正常，你会看到类似如下的输出：

2025-10-21T09:44:03Z INF Start Websocket listener host=127.0.0.1:25565

保持终端窗口运行（或者你也可以在 screen 里运行 cloudflared），启动 Minecraft，进入多人游戏，添加服务器，地址填写：

127.0.0.1

完成即可。然后现在就可以连接服务器了。

向导：如何注册或登入服务器账号

由于没有开启 online mode，为确保服务器的安全，本服务器使用 AuthMe 插件来验证玩家身份。在启动器中添加离线账号启动游戏并连接服务器后，AuthMe 会要求玩家使用如下命令注册你的账号：

/register 密码 密码

注意：密码不可包含空格，且至少需要 6 个字符。每个 IP 最多注册 3 个账号。

注册成功后，AuthMe 会要求玩家重新登入一次。你需要使用如下命令来登入账号：

/login 密码

如需修改密码，可以使用如下命令来修改密码：

/changepassword 旧密码 新密码 新密码

关于 AuthMe 的更多命令和用法可以参阅 GitHub 上的项目文档。

向导：使用自定义皮肤

本服务器使用 SkinsRestorer 插件来支持自定义皮肤。

准备好一个 .png 皮肤文件，并把文件上传到 imgur.com，上传成功后，右键单击图片本身来拷贝图片链接，拷贝的链接需要是看起来像这样 https://i.imgur.com/5GMdSMc.png

然后回到游戏，使用如下命令来设置皮肤，[url] 相应的替换为你拷贝的链接：

/skin url [url]

过一会儿就可以看到皮肤生效了。
