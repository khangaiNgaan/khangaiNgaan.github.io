README caffeine-Ink / Server of Neko

Server of Neko 是一个 Minecraft Java Edition 服务器，目前使用的服务端版本是 1.21.8。

服务器地址(端口默认)：

mc.caffeine.ink

本服务器目前加载的模块有:

AuthMe              #用于验证玩家身份，用法参见向导
SkinsRestorer       #用于自定义皮肤，用法参见向导
SinglePlayerSleep   #单个玩家睡眠即可跳过夜晚或雷暴

向导：如何加入游戏

你可以使用 Microsoft Windows / macOS / Android 游玩。

对于 Windows / macOS 用户，可以使用 HMCL 启动器或其他启动器安装 Java 版 1.21.8 进行游玩。HMCL 是一个开源的 Minecraft Java Edition 启动器，你可以通过下面的链接访问 HMCL 官网，下载稳定版 HMCL，或是查看 HMCL 的官方帮助文档。

HMCL 官方网站
HMCL 帮助文档
HMCL GitHub 项目地址

对于 Android 用户，可以使用 FCL 启动器安装 Java 版 1.21.8 进行游玩。

Foldcraft Launcher GitHub 项目地址
Foldcraft Launcher 下载页面

这些启动器都会自动安装合适的 Java 环境。

本服务器没有开启正版验证（online mode），因此你可以使用离线账号加入服务器游玩。

向导：如何注册或登入服务器账号

由于没有开启 online mode，为确保服务器的安全（2025年4月有过被岩浆山炸服的惨痛教训），本服务器使用 AuthMe 插件来验证玩家身份。在多人游戏中添加本服务器并连接后，AuthMe 会要求玩家使用如下命令注册账号：

/register 密码 密码

注意：密码不可包含空格，且至少需要 6 个字符。

注册成功后，AuthMe 会要求玩家重新登入一次。你需要使用如下命令来登入账号：

/login 密码

如需修改密码，可以使用如下命令来修改密码：

/changepassword 旧密码 新密码 新密码

关于 AuthMe 的更多命令和用法可以参阅 GitHub 上的项目文档。

向导：如何使用自定义皮肤

本服务器使用 SkinsRestorer 插件来支持自定义皮肤。

准备好一个 .png 皮肤文件，并把文件上传到 imgur.com，上传成功后，右键单击图片本身来拷贝图片链接，拷贝的链接需要是看起来像这样 https://i.imgur.com/5GMdSMc.png

然后回到游戏，使用如下命令来设置皮肤，[url] 相应的替换为你拷贝的链接：

/skin url [url]

过一会儿就可以看到皮肤生效了。
